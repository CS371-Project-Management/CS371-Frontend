## Getting Started

--- FROM ( SUN ) ---

first
```bash
npm install
```

next, run this command
```bash
npm install framer-motion lucide-react
```

then, to run program on port:3000
```bash
npm run dev
```

--- END ( SUN ) ---

first
```bash
npmrun
```
build image
```bash
docker compose build
```
run docker container
```bash
docker-compose up -d
```
